# Tech Stack

## Core Technologies
- **Framework:** [Next.js](https://nextjs.org/) (App Router) - High performance, SEO-friendly, and seamless page transitions.
- **Language:** [TypeScript](https://www.typescriptlang.org/) - Robust type safety for a professional, scalable codebase.
- **Styling:** [CSS Modules](https://github.com/css-modules/css-modules) / Vanilla CSS - Ensuring full control over custom aesthetics and performance.
- **Animations:** [Framer Motion](https://www.framer.com/motion/) - The gold standard for complex, ultra-smooth animations in React.

## Tools & Libraries
- **Icons:** [Lucide React](https://lucide.dev/) - Modern, lightweight icon set.
- **State Management:** [Zustand](https://github.com/pmndrs/zustand) - Minimalist, high-performance state management.
- **Fonts:** Custom Fonts (extracted from the redesign HTML file) via Next.js `next/font`.

## Infrastructure
- **Hosting:** [Vercel](https://vercel.com/) - Optimizes Next.js performance and provides global edge network delivery.
- **Version Control:** Git / GitHub.
